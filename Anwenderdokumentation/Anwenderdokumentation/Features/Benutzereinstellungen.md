# Benutzereinstellungen

## Überblick
In den Benutzereinstellungen können Sie Ihre persönlichen Informationen verwalten, Ihr Passwort ändern und Ihr Konto absichern oder löschen.

## Profil verwalten
Hier können Sie Ihre Stammdaten einsehen und aktualisieren:
- **Name:** Ihr Vor- und Nachname.
- **E-Mail-Adresse:** Die für die Anmeldung und Benachrichtigungen verwendete Adresse.

## Sicherheit & Passwort
Um Ihr Konto zu schützen, können Sie Ihr Passwort jederzeit ändern.

### Passwort-Anforderungen
Ein sicheres Passwort muss folgende Kriterien erfüllen:
- Mindestens **8 Zeichen** lang.
- Mindestens ein **Großbuchstabe**.
- Mindestens ein **Kleinbuchstabe**.
- Mindestens eine **Zahl**.
- Mindestens ein **Sonderzeichen** (z. B. !, @, #, $).

## Abmeldung (Logout)
Um sich sicher vom System abzumelden, klicken Sie im Benutzermenü auf **Abmelden**. Dies ist besonders wichtig, wenn Sie einen öffentlichen oder geteilten Computer verwenden.

## Konto löschen
Wenn Sie GoalDone nicht mehr nutzen möchten, können Sie Ihr Konto dauerhaft löschen.
1. Navigieren Sie zu den Einstellungen.
2. Wählen Sie den Bereich **Konto löschen**.
3. Zur Bestätigung müssen Sie das Wort **LÖSCHEN** in das dafür vorgesehene Feld eingeben.
4. Bestätigen Sie den Vorgang.

**Achtung:** Dieser Vorgang kann nicht rückgängig gemacht werden. Alle Ihre persönlichen Daten werden dauerhaft entfernt.

## Einschränkungen
- **Letzter Administrator:** Wenn Sie der letzte Administrator einer Organisation sind, können Sie Ihr Konto erst löschen, nachdem Sie ein anderes Mitglied zum Administrator ernannt haben. Falls Sie Goaldone als Organisation nicht mehr nutzen wollen, dann melden Sie sich bitte an den Super-Admin / Kundenservice.

## Context Notes

**Professional Tone:** Use formal "Sie" address in transactional emails. Avoid informal "du" unless explicitly requested by user preference.

**Capitalization:** All nouns capitalized (German grammar rules). Article-noun combinations: "das Konto", "die Verknüpfung", "die E-Mail-Adresse".

**Word Choices:**
- "Verknüpfung" specifically for account linking (not "Verbindung" which means general connection)
- "E-Mail-Adresse" for email address (hyphenated per German style)
- "Passwort zurücksetzen" not "Passwort ändern" (change vs. reset distinction)
- "Konto" for user account (not "Benutzer" which is person)

## Usage

All email templates and HTML pages in Phase 5.3 must use terms from this glossary. No ad-hoc translations. If new terms needed, add to this document and use consistently thereafter.

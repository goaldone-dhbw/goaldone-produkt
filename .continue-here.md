## The Last Problem (READY TO FIX)

**Issue:** `ConstraintViolationException: getMyAccounts.xOrgID: darf nicht null sein`

**Root Cause Identified:**
- TenantContextFilter stores validated org-ID in `TenantContext.set(xOrgId)` ✓
- But UserAccountsController tries to read from `@RequestHeader("X-Org-ID")` → gets NULL ❌
- Mismatch between filter storage and controller parameter binding

**Solution (Ready to Implement):**
Controller should read from TenantContext instead of HTTP header:

```java
// Current (BROKEN):
@GetMapping("/users/accounts")
public AccountListResponse getMyAccounts(@RequestHeader("X-Org-ID") UUID xOrgID)

// Should be:
@GetMapping("/users/accounts")
public AccountListResponse getMyAccounts() {
  UUID xOrgID = TenantContext.get();  // Filter already validated and stored it here
  ...
}
```

Files to modify:
- `backend/src/main/java/de/goaldone/backend/controller/UserAccountsController.java` (getMyAccounts method)
- Any other controller methods using `@RequestHeader("X-Org-ID")`

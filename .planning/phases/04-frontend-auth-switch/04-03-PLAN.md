  <name>Task 1: Implement on-demand token refresh in authInterceptor</name>
  <files>frontend/src/app/core/auth/auth.interceptor.ts</files>
  <action>
Enhance the authInterceptor to check token expiry before every request and refresh if needed. This replaces the silent refresh approach with explicit per-request validation.

Implementation steps:

1. **Add token expiry buffer check:**
   ```typescript
   private isTokenExpirySoon(expiresAt: number): boolean {
     const bufferMs = 5 * 60 * 1000; // 5 minute buffer
     return (expiresAt - Date.now()) < bufferMs;
   }
   ```
   Extract `expires_at` from decoded token. If token expires in less than 5 minutes, trigger refresh.

2. **Add refresh logic before request:**
   ```typescript
   export const authInterceptor: HttpInterceptorFn = (req, next) => {
     const authService = inject(AuthService);
     
     // 1. Check if we have a valid access token
     if (!authService.hasValidAccessToken()) {
       // No token at all, let request go through (will get 401)
       return next(req);
     }
     
     // 2. Check if token will expire soon
     const decodedToken = authService.getDecodedAccessToken();
     if (decodedToken && authService.isTokenExpirySoon(decodedToken.exp * 1000)) {
       // Refresh token synchronously before continuing
       return authService.refreshToken().pipe(
         switchMap(() => proceedWithAuthorization(req, authService, next)),
         catchError(() => {
           // Refresh failed, continue anyway (backend will handle 401)
           return proceedWithAuthorization(req, authService, next);
         })
       );
     }
     
     // 3. Token is valid, proceed with authorization
     return proceedWithAuthorization(req, authService, next);
   };
   
   function proceedWithAuthorization(
     req: HttpRequest<any>,
     authService: AuthService,
     next: HttpHandlerFn
   ): Observable<HttpEvent<any>> {
     const token = authService.getAccessToken();
     const apiBasePath = (window as any).__env?.['apiBasePath'] || 'http://localhost:8080';
     
     if (token && req.url.startsWith(apiBasePath)) {
       // Inject Bearer token
       req = req.clone({
         setHeaders: { Authorization: `Bearer ${token}` },
       });
     }
     
     return next(req);
   }
   ```

3. **Make isTokenExpirySoon() method accessible:**
   In AuthService, add a public method:
   ```typescript
   isTokenExpirySoon(): boolean {
     const decodedToken = this.getDecodedAccessToken();
     if (!decodedToken || !decodedToken.exp) {
       return false;
     }
     const bufferMs = 5 * 60 * 1000; // 5 minute buffer
     return (decodedToken.exp * 1000 - Date.now()) < bufferMs;
   }
   ```

Per D-03: On-Demand Token Refresh and D-14: Per-Request Refresh Check.

   function resolveOrgIdForRequest(
     orgContextService: OrgContextService
   ): string | null {
     // Priority: dialog org > settings org > default org
     let orgId = orgContextService.getDialogOrg();
     if (!orgId) {
       orgId = orgContextService.getSettingsOrg();
     }
     if (!orgId) {
       const defaultOrg = orgContextService.getDefaultOrg();
       orgId = defaultOrg?.id || null;
     }
     return orgId;
   }

  <action>
Add methods to AuthService for token refresh and revocation. These are called by the interceptor and logout() respectively.

1. **Add refreshToken() method:**
   ```typescript
   refreshToken(): Observable<boolean> {
     if (!this.oauthService.getRefreshToken()) {
       return of(false); // No refresh token available
     }
     
     return new Observable(observer => {
       this.oauthService.refreshToken().then(
         () => {
           this.logger.info('Token refreshed successfully');
           observer.next(true);
           observer.complete();
         },
         error => {
           this.logger.error('Token refresh failed:', error);
           observer.next(false);
           observer.complete();
         }
       );
     });
   }
   ```

   Alternative (if angular-oauth2-oidc supports promises):
   ```typescript
   async refreshToken(): Promise<boolean> {
     try {
       await this.oauthService.refreshToken();
       return true;
     } catch (error) {
       this.logger.error('Token refresh failed:', error);
       return false;
     }
   }
   ```

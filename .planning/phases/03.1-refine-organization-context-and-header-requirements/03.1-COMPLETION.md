## Architecture Improvements

### Backend Changes
- **TenantContext:** ThreadLocal-based org ID storage per request
- **TenantContextFilter:** Request-scoped validation and header extraction
- **CurrentUserResolver:** TenantContext-aware user/membership resolution
- **Security Expressions:** Per-org RBAC via hasOrgRole() custom expressions

### Frontend Changes
- **TenantService:** Angular Signal-based reactive org state management
- **TenantInterceptor:** Automatic X-Org-ID header injection for all requests
- **AuthService:** Membership extraction from JWT for UI-driven org selection
- **HTTP Context:** Support for header override per request

### Database Changes
- **User vs. Membership:** Clear separation between identity and org-specific accounts
- **auth_user_id:** Unified user identifier (indexed for login performance)
- **Composite UNIQUE:** One membership per user per organization

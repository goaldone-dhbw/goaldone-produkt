Full rewrite of `MemberInviteService.java`:

**Class structure:**
```java
@Slf4j
@Service
@RequiredArgsConstructor
public class MemberInviteService {
    private final AuthServiceManagementClient authServiceClient;
    private final MembershipRepository membershipRepository;
    private final OrganizationRepository organizationRepository;
}
```
Remove: `ZitadelManagementClient`, `@Value` fields for `goaldoneProjectId`/`mainOrgId`, `UserService` (if only used for Zitadel lookup).

**inviteMember(UUID orgId, String email, Role role)** — per D-06, D-07:
1. Get caller UUID from SecurityContext (`getCallerSub()`)
2. Call `authServiceClient.createInvitation(orgId, email, UUID.fromString(callerSub), role)`
3. On success: create `MembershipEntity` with:
   - `organizationId = orgId`
   - `status = MemberStatus.INVITED`
   - `invitationId = response.getId()` (UUID from auth-service)
   - `authUserId = null` (filled on acceptance via JIT)
   - `role = role`
4. Save entity to `membershipRepository`
5. On auth-service error: throw `ResponseStatusException(BAD_GATEWAY, "Failed to create invitation")`

**reinviteMember(UUID orgId, UUID membershipId)** — per D-08:
1. Find existing `MembershipEntity` by ID; verify status = INVITED
2. Call `authServiceClient.createInvitation(orgId, entity.getEmail(), callerSub, entity.getRole())`
3. Update `entity.setInvitationId(response.getId())`
4. Save entity

**cancelInvitation(UUID orgId, UUID membershipId)** — per D-08:
1. Find existing `MembershipEntity` by ID; verify status = INVITED
2. Call `authServiceClient.cancelInvitation(entity.getInvitationId())`
3. Delete the `MembershipEntity` (or mark as CANCELLED — prefer delete for simplicity)
4. On auth-service error: throw `ResponseStatusException(BAD_GATEWAY, ...)`

All methods wrap auth-service errors:
```java
try {
    // auth-service call
} catch (AuthServiceManagementException e) {
    log.error("Auth-service error: {}", e.getMessage());
    throw new ResponseStatusException(HttpStatus.BAD_GATEWAY, "Auth service error: " + e.getMessage());
}
```

**MemberManagementService.java** — Full rewrite:

Class structure:
```java
@Slf4j
@Service
@RequiredArgsConstructor
public class MemberManagementService {
    private final AuthServiceManagementClient authServiceClient;
    private final MembershipRepository membershipRepository;
    private final OrganizationRepository organizationRepository;
    private final MembershipDeletionService membershipDeletionService;
}
```
Remove: `ZitadelManagementClient`, `UserService` (if only used for Zitadel lookup), `@Value` fields for `goaldoneProjectId`/`mainOrgId`.

**listMembers(UUID orgId)** — per D-09, D-10:
1. Call `authServiceClient.getMembers(orgId)`
2. Map `List<AuthMemberResponse>` to `List<MemberResponse>`:
   - `userId = item.getUserId()`
   - `email = item.getEmail()`
   - `firstName = item.getFirstName()`
   - `lastName = item.getLastName()`
   - `role = item.getRole()`
   - `status = item.getStatus()` (ACTIVE or INVITED)
3. Return the mapped list

**changeMemberRole(UUID orgId, UUID userId, Role newRole)** — per D-18, D-19:
1. Call `authServiceClient.updateMembershipRole(userId, orgId, newRole)` FIRST
2. Catch `AuthServiceManagementException`:
   - If `e.getStatusCode() == 409` → throw `new ResponseStatusException(HttpStatus.CONFLICT, "LAST_ADMIN_CANNOT_BE_DEMOTED")`
   - Otherwise → throw `new ResponseStatusException(HttpStatus.BAD_GATEWAY, "Auth service error")`
3. On success: update local `MembershipEntity` role (find by userId + orgId, set new role, save)

**removeMember(UUID orgId, UUID userId)** — per D-18, D-19:
1. Call `authServiceClient.deleteMembership(userId, orgId)` FIRST
2. Catch `AuthServiceManagementException`:
   - If `e.getStatusCode() == 409` → throw `new ResponseStatusException(HttpStatus.CONFLICT, "LAST_ADMIN_CANNOT_BE_REMOVED")`
   - Otherwise → throw `new ResponseStatusException(HttpStatus.BAD_GATEWAY, "Auth service error")`
3. On success: delete local `MembershipEntity` (find by userId + orgId)
